Code the Gong is a Wollongong based CoderDojo. CoderDojo is a global movement run by volunteers to help teach young people to code. Visit the dojo to learn to code, build a website, create an app and explore technology.

The CoderDojo movement believes that an understanding of programming languages is increasingly important in the modern world, that it’s both better and easier to learn these skills early, and that nobody should be denied the opportunity to do so.
